# Credits Calculator
## Features

- Edit course grades, types, and credits to simulate your GPA.
    
- Set target credits for different course categories.
    
- Drag and drop summary categories to rearrange their order.
    
- Export your customized report as a clean, print ready PDF.
    
- Share category setups and course suggestions with batchmates using templates.
    

## Usage Guide

### 1. Importing the Base HTML

To load your actual data into the tool:

1. Log in to the AIMS portal and get your academic report with [IITH AIMS Helper 2.0 - Chrome Web Store](https://chromewebstore.google.com/detail/iith-aims-helper-20/imkiafnadcmlcbahbklilfgenfknniin) extention.
    
2. Press Ctrl+S (or Cmd+S) and save the page as "Webpage, HTML Only".
    
3. Open the website, click "Update Base HTML Report", and upload the saved file.
    

### 2. Using Templates (Export / Import)

Templates allow you to share custom course types, category sorting orders, target credit requirements, and suggested types for specific courses.

- **Export Template**: Click this button to download your current setup as a `.json` file.
    
- **Import Template**: Click this button and select a `.json` file to instantly apply a shared setup to your report. Suggested course types will appear grouped in your dropdown menus, and a small asterisk will flag courses that do not match the template's suggestion.
    

### 3. Controls

- **Undo Edit**: Reverts your most recent modification.
    
- **Reset Edits**: Removes all manual changes (like altered grades or types) while keeping your base HTML data intact.
    
- **Clear All Data**: Completely wipes all saved data, templates, and edits, returning the tool to a blank state.
